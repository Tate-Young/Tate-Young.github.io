## 如何控制曝光

以下采用[知乎蜜茶摄影的撰文](https://www.zhihu.com/question/21427664) 👈

### 光圈

![光圈](/style/images/photography/aperture.jpg)

**光圈**就是用来控制相机进光量多少的装置，当我们在拍照的时候，光线进入到相机内部第一道门槛就是光圈。光圈的开口越大，进入相机的光线就越多，即照片越亮；反之，就越少。在相机中，光圈用 `F+数字` 的形式表示。**F 后面的数字越大，表示光圈越小**，其公式为: `光圈系数值 = 焦距 / 光圈孔径`。举个栗子:

![光圈示例 1](/style/images/photography/aperture-1.jpg)

另一方面光圈可以用来制作**景深**，可以简单理解为一张照片中清晰的部分。照片中，**清晰的部分越多，表示景深越深**。举个栗子:

![光圈示例 2](/style/images/photography/aperture-2.jpg)

则说明**光圈越大，虚化越严重，景深越浅**，这样我们去拍照想突出某一事物的时候，可以选择大光圈:

![光圈示例 3](/style/images/photography/aperture-3.jpg)

如果你想在晚上拍路灯的星芒效果，可以采用小光圈:

![光圈示例 4](/style/images/photography/aperture-4.jpg)

> 根据前转盘可以快捷调节光圈大小

> 不同焦距下，可设置的最低光圈系数可能不同，可以从镜头参数中读取信息

### 快门

很多人都以为拍照时按下的快门按钮就是**快门**，其实这只是一个触发装置，在相机中还有一个专门的**快门帘**组件。相机中一般用 `1/400` 这样的形式表示快门时间(也叫快门速度)。**这个数值越大，快门帘开启的时间越长，进入相机的光线就越多，即照片就越亮**；反之数值越小，进光量越少:

![快门示例 1](/style/images/photography/shutter-1.jpg)

快门会影响记录的"轨迹": 可以定格瞬间，也可以记录过程:

![快门示例 2](/style/images/photography/shutter-2.jpg)

但是在手持拍摄的时候，很容易拍模糊。这是因为在快门开启的时间内，只要手抖了，哪怕非常轻微，也会被相机记录下来，导致拍出的照片模糊。为了避免这种情况，建议大家记住一个概念：**安全快门**，它要求快门时间小于焦距的倒数，比如用 200mm 焦距拍照，那快门建议小于 1/200 秒，这样拍出的照片才不会模糊。

> 安全快门仅是一个参考数值，如果你手持抖成马，那谁也帮不了你。而且使用三脚架等防抖工具就直接可以忽略它了。

> 根据后转盘可以快捷调节光圈大小

### 感光度 ISO

**感光度**又称为 **ISO**，是指相机对光线的敏感程度，一般用 `ISO 100` 表示，**数字越大，表明设置的感光度越大，相机对光线越敏感**。在光圈和快门不变的情况下，ISO 越高，相机的感光能力越强，拍出来的照片就会越亮；反之就会越暗:

![ISO 示例 1](/style/images/photography/iso-1.jpg)

但是平时我们基本不会用提高感光度来增强画面亮度的，因为高 ISO 会让画面看起来有颗粒感，这些颗粒就是所谓的噪点，所以理论上 ISO 越低越好:

![ISO 示例 1](/style/images/photography/iso-2.jpg)

### 曝光

**曝光**是指在摄影过程中进入镜头照射在感光元件上的光量，一张照片曝光是否正常，就是由上述三者共同决定的，在相机中的一般都有曝光刻度表:

* 当指针指在中间的"0"时 - 表示曝光正常，拍出来的照片亮度合适；
* 当指针指在左边的负数区 - 表示曝光不足，拍出来的照片偏暗；
* 当指针指在右边的正数区 - 表示曝光过度，拍出来的照片偏亮。

![曝光示例 1](/style/images/photography/exposure-1.jpg)

那么他们之间是怎么协同合作的呢? 我们以下图水龙头为例：如果拍一张曝光正常的照片是接满一桶水，我们用水龙头阀门的大小代表光圈，用水流时间代表快门，ISO 代表滤网:

![曝光示例 2](/style/images/photography/exposure-2.jpg)

假定 ISO 不变的情况下，如果把水龙头阀门拧小，那么接满一桶水的时间就会变长；如果把水龙头拧大，那接满一桶水的时间就会缩短。所以光圈和快门是此消彼长的动态平衡关系。ISO 起到滤网的作用，ISO 越大，表示滤网的空隙越大，出水量就越大，但是杂质也会越多，拍出来的照片就会有很多的颗粒噪点。反之，ISO 越小，滤网的空隙越小，杂质就越少，照片也就越干净。

把这个例子应用于我们平时的拍摄中：**如果需要慢快门操作的时候，比如记录运动轨迹，可以缩小光圈，延长快门时间**；如果需要大光圈操作的时候，比如拍摄背景虚化，可以加大光圈，缩短快门时间:

![曝光示例 3](/style/images/photography/exposure-3.jpg)

## 相机相关术语与参数

### 屈光度

光线由一种物体射入到另一种光密度不同的物质时，其光线的传播方向产生偏折，这种现象称为**屈光现象**，表示这种屈光现象大小即屈光力的单位就是**屈光度**，缩写为“D”。在配近视眼镜的时候经常用到，而相机上的屈光度，也跟眼睛的近视度数有关。简而言之，不同视力状况的童鞋要保持当前取景器的画面观察清晰，就必须调节适合自己的屈光度。有个简单的调节方式，可以半按快门使得取景器的参数显示出来，然后调节屈光度旋钮，让字体最清晰即可。

### 对焦 / 合焦

* **对焦**是使相机能清晰成像的过程，分为自动对焦和手动对焦
* **合焦**则是结果，指在传感器上形成高锐度清晰的图像
* **跑焦**是对应合焦来说的，通常是指在我们完成对焦后由于各种原因而导致的焦点变化或焦点不实，从而画面较为模糊。

### 宽容度 / 动态范围

* **宽容度**是相机记录最亮和最暗细节与层次反差的能力。一般说来胶片的宽容度应该是越大越好。宽容度小的胶片，常会使景物明、暗部分在影像上得不到正确反映，损害影像的真实性。
* **动态范围**是代表着一张照片高光到暗部的范围，动态范围越大，宽容度越高

![宽容度](/style/images/photography/camera-1.jpeg)

> **高动态范围图像（High-Dynamic Range，简称 HDR）**，相比普通的图像，可以提供更多的动态范围和图像细节，根据不同的曝光时间的 LDR(Low-Dynamic Range)，并利用每个曝光时间相对应最佳细节的 LDR 图像来合成最终 HDR 图像，它能够更好的反映出真实环境中的视觉效果。

### 对比度 / 饱和度

* **对比度** - 指的是一幅图像中明暗区域高光和暗部之间不同亮度层级的测量，差异范围越大代表对比越大。一般来说对比度越大，图像越清晰醒目，色彩也越鲜明艳丽；而对比度小，则会让整个画面都灰蒙蒙的
* **饱和度(Saturation)** - 指的是色彩的浓度，饱和度越高色彩越浓，

### 帧率 / 码率

* **帧率** - 以帧为单位的位图图像连续出现在显示器上的频率(Hz)。一般来说 30fps 就是可以接受的，但是将性能提升至 60fps 则可以明显提升交互感和逼真感，但是一般来说超过 75fps 就不容易察觉到有明显的流畅度提升了。如果帧率超过屏幕刷新率只会浪费图形处理的能力，因为监视器不能以这么快的速度更新，这样超过刷新率的帧率就浪费掉了。
* **帧数** - 为帧生成数量的简称，每一帧都是静止的图象，快速连续地显示帧便形成了运动的假象，因此高的帧率可以得到更流畅、更逼真的动画。`帧率 =  帧数 / 时间`，单位为帧每秒 fps。因为人眼的视觉残留特性是由视神经元的反应速度造成的，其时值是 1/24 秒，肉眼在看超过 24fps 的静态图片就会认为是连续动态视频，是动画、电影等视觉媒体形成和传播的根据。
* **码率** - 即比特率，是指每秒传送的比特(bit)数，单位为 bps(Bit Per Second)。比特率越高，每秒传送数据就越多，画质就越清晰。通俗一点的理解就是取样率，单位时间内取样率越大，精度就越高，处理出来的文件就越接近原始文件

> FPS（指 FPS 射击游戏）重在 FPS（指帧率）

> **升格**：即慢动作，通过 S&Q 档位可以拍摄，比如 100ps 25p(4 倍慢速)；也可以提前通过普通视频档位拍摄 100ps 视频，后期通过慢速处理。反之则为降格。

### SD 存储卡 / 记忆棒

* **SD 存储卡(Secure Digital Memory Card)**是由松下电器、东芝和 SanDisk 联合推出的一种基于半导体快闪记忆器的新一代记忆设备，由于它体积小、数据传输速度快、可热插拔等优良的特性，被广泛地于便携式装置上使用，例如数码相机、平板电脑和多媒体播放器等
* **记忆棒(Memory Stick)**是由索尼推出的专属移动存储媒体，目前只能用在索尼自家的产品上，分别有四种类型：`Memory Stick、Memory Stick DUO、Memory Stick PRO、Memory Stick PRO DUO`
* **Micro SD 存储卡**原名即 **TF(Trans-flash) 卡**，由 SanDisk 推出，主要运用于移动设备存储。它能通过 SD 转接卡来接驳于 SD 卡插槽中使用

> 说到这些存储卡，不得不提到日常用的最多的**SIM 卡**，它主要完成两种功能：存储数据(控制存取各种数据)和在安全条件下(个人身份号码 PIN、鉴权钥 Ki 正确)完成客户身份鉴权和客户信息加密算法的全过程。GSM 系统通过 SIM 卡来识别用户，可以提供上网、打电话等服务

那么我们看到的 SD 存储卡的 **UHS-I** 与 **UHS-II** 又有什么区别呢？"UHS" 就是“Ultra-High Speed，超高速”的缩写，支持超高速总线接口的存储卡一般都会有着更快的读写速度，这一标准是由“SD 卡协会”这一非营利性组织来制定的。UHS-I（第一代超高速）的 SD 卡产品于 2010 年被发布，理论传输速度可达 104MB/s，而支持 UHS-II（第二代超高速） 的 SD 卡产品理论传输速度可达 312MB/s，启用了额外一排的金手指。目前我们在选购这类高速 SD 存储卡的时候，应该注意我们的设备是否支持这一接口。

### 热靴

**热靴(Hot Shoe)**是直接意译而来的，也叫燕尾槽。它是各种数码影像器材连接各种外置附件的一个固定接口槽，可连接的附件包括闪光灯、GPS 定位器、摄像灯以及麦克风等等。

### 耶稣光

**耶稣光**也叫云隙光，是一种丁达尔效应，即从云雾的边缘射出的阳光，照亮空气中的灰尘而使光芒清晰可见。许多电影、画作、动漫画也常使用洒落地面的云隙光作为神圣、崇高、救赎的象征。

### 全画幅 / 半画幅

最大的区别就是其中相机的核心传感器尺寸，**全画幅**指的是保持 36×24mm 规格的胶片大小，指代使用这个规格尺寸的传感器，而**半画幅**相机的感光元件尺寸为 18mm*24mm。相对而言，传感器尺寸越大，细节也就越丰富，画质也更好，因此全画幅相机在拍照时的画面质量、细节与感光度范围都比半画幅要出色。

### 对焦模式与区域

对焦模式可以根据被摄物体的移动状况选择对焦模式:

| 对焦模式 AF      |   含义   |  描述 |
| ------------ | ------- | --- |
| AF-S | 单次 | 在合焦时固定焦点，适合拍摄静物 |
| AF-A | 自动 | 结合上下两种特性，会根据被摄体状态自动设置固定或者连续对焦。连拍时，第二张以后自动切换为连续 AF |
| AF-C | 连续 | 对移动中的被摄体连续对焦 |
| MDF | 直接手动对焦 | 结合自动和手动，可以在自动对焦后手动微调。对微距拍摄较为方便 |
| MF | 手动对焦 | 纯手动调节对焦环进行对焦 |

> 默认使用 C3 自定义按钮快捷呼出对焦模式

对焦区域则用来改变对焦位置，用于难以合焦等情况:

| 对焦区域        |   描述   |
| ------------ | ------- |
| 广域 | 以显示屏整体为基准自动对焦 |
| 区 | 以所选区为基准自动对焦 |
| 中间 | 以显示屏中央区域为基准自动对焦 |
| 自由点 | 以所选小范围区域为基准自动对焦，可从小到大调节为 S、M、L |
| 扩展自由点 | 有两个嵌套选取，以内部为优先，无法合焦时会尝试再用外部区域进行对焦 |
| 锁定 AF | 只有在对焦模式设置为 AF-C 才可用。即选择连续对焦时所采用的对焦区域类型 |

> 默认使用 C2 自定义按钮快捷呼出对焦区域

### 拍摄模式

一般我们可以根据被摄体和拍摄目的来改变照相模式:

| 拍摄模式        |   含义   | 描述 |
| ------------ | ------- | --- |
| AUTO | 智能自动 | 自动识别场景 |
| P | 程序自动 | 自动设定曝光，包括快门和光圈。其他可以手动调整 |
| A | 光圈优先 | 想要虚化背景时，设定光圈值进行拍摄 |
| S | 快门优先 | 想要抓拍时，设定快门速度进行拍摄 |
| M | 手动曝光 | 纯手动。**通过手动调节三要素，让曝光提示趋近于 0，才能尽可能正确的曝光** |
| B | B 门拍摄 | 可以用长时间曝光拍摄移动轨迹 |
| 1/2 | 调出内存 | 注册自定义档位 |
| SCN | 场景选择 | 选择适合的环境进行拍摄，比如肖像、夜景、风景、微距等 |
| 视频 | 动态影像 | 设定动态影像的曝光模式并拍摄。一般拍摄视频时，**快门速度设置为帧率的两倍分之一** |
| S&Q | 慢和快动作 | 可拍摄快动作和慢动作动态影像，但无法录制声音，码率较低 |

![拍摄模式](/style/images/photography/camera-5.jpg)

> 注意没有现成 B 档，需要先调到 M 档，然后使用后转盘调节到 **BULB**。一般情况下拍摄烟花等时，可以使用小光圈，低 ISO(曝光时间长的话画面内的噪点容易变得明显)，对焦模式可以使用 DMF 或 MF，焦距为无穷远，由于快门时间较长，建议使用三脚架。另一个需要注意的是 B 档测光系统不工作。

### 测光与中性灰标准反推测

测光模式一般是设定相机自动决定曝光时，用显示屏的哪个部分测光，直接用 Fn 功能可以快捷呼出:

* 多重 - 对分割成多个区域的显示屏的各区域进行测光，以确定画面整体的最佳曝光。可以设置"多重测光时人脸优先"，相机会以检测到的人脸信息为基准进行测光
* 中心 - 将重点放在显示屏中央的同时，对整体的亮度进行测光
* 点测光 - 只会对点测光圆内进行测光，可以设置`标准`和`大`来改变测光圆的大小。在菜单栏中设置"点测光点 --> 对焦点联动"(默认为中间)，测光圆可以随着焦点变动位置
* 整个屏幕平均 - 对画面整体进行平均测光
* 强光 - 对画面内的高光部分重点进行测光，比较适合一些有强光照射的画面

相机的测光系统计算曝光量采用的是**中性灰标准反推测**，意思是相机会以中性灰色作为它的测光标准，反推出需要的曝光量。比如相机会获取到拍摄画面，然后做去色处理，再把得到的这一个黑白画面平均一下灰度，再将这个灰度和中性灰做对比，如果灰色程度一样，则会给到标准的曝光量，即曝光正常；若要深，则相机会相应增加光亮，直到画面灰度与中性灰程度一致为止，这样最终得到的画面会过曝；若浅则反之。

相机获取的这个拍摄画面区域就是由我们上面说的测光来决定的。举个例子，如果我们使用点测光对准一个黑色物体，而周围环境即使很亮，但是测光系统也只会对测光圆内的画面进行提取，这样提取出来的灰度会较深，因此整体拍摄的画面会过亮。所以如果我们使用相机的半自动拍摄模式进行拍摄的时候，曝光三要素里的部分参数是由相机的测光系统赋值的，但是在一些极端的场景下，比如拍摄雪景，不管是采用哪种测光模式，都会造成画面偏暗，因此会用到我们下面讲到的曝光补偿。

### 曝光补偿

**曝光补偿**一种曝光控制方式，拍摄者可以根据自己的想法调节照片的明暗程度，创造出独特的视觉效果，一般来说就是有意识地变更相机自动演算出的“合适”曝光参数。拍摄环境比较昏暗，进行曝光补偿的时候，如果照片过暗，要增加 EV 值，EV 值每增加 1.0，相当于摄入的光线量增加一倍。由于相机的快门时间或光圈大小是有限的，因此并非总是能达到 2EV 的调整范围，因此曝光补偿也不是万能的，在过于暗的环境下仍然可能曝光不足，此时要考虑配合闪光灯或增加相机的 ISO 感光灵敏度来提高画面亮度。

上面有提到拍摄雪景这种场景，就是被拍摄的白色物体在照片里看起来是灰色或不够白的时候，要增加曝光量，简单的说就是“**白加黑减**”，看起来与曝光的基本原则和习惯是背道而驰的，其实不然，这是因为相机的测光往往偏重于中心主体，白色的主体会让相机误以为环境很明亮，因而造成曝光不足。

> 只有在使用了半自动模式才需要调节曝光补偿，手动模式时显示屏该位置会变为曝光提示

### 白平衡与色温

许多人在使用相机拍摄的时候都会遇到这样的问题：在日光灯的房间里拍摄的影像会显得发绿，在室内钨丝灯光下拍摄出来的景物就会偏黄，而在日光阴影处拍摄到的照片则莫名其妙地偏蓝，其原因就在于**白平衡**的设置上。其基本概念是“不管在任何光源下，都能将白色物体还原为白色”。

在物理学上，温度越高，光线中的蓝色光线较多，反之则红色光线较多。在相机的白平衡模式选择中，我们可以看到它分为了几种场景：日光、阴影、阴天、白炽灯、色温/滤光片和自定义等等。而且每一种场景都对应了一种参数，即**色温**，它是表示光线中包含颜色成分的一个计量单位，当加热到一定的温度，[黑体](https://baike.baidu.com/item/黑体/5398327)发出的光所含的光谱成分，就称为这一温度下的色温，计量单位为“K”（开尔文）。色温/滤光片可以让我们去自定义色温，让相机来模拟我们所处的一个环境。需要注意的是 5500K 是一个分水岭，在此之上相机会判断色温较高，即蓝色光线较多，因此会利用红色光线来进行中和，反之则利用蓝色光进行中和。

我们来看下两个场景，一个是拍摄日出，当我们利用 M 档来拍摄时，建议关闭自动白平衡或者将色温调高，因为相机会检测到当前温度较低，即色温较低，然后利用蓝色光线来中和；另一个是拍摄蓝天，如果我们将色温调低，相机会利用更多的蓝色来中和，因此拍摄出来的天空会更蓝。

> 默认使用 C1 自定义按钮快捷呼出白平衡

### PP 值

在图片配置文件选项里，我们可以设置 PP1 - PP10，PP 代表了 **图像配置文件(Picture Profile)**，接下来我们看看他们的一些区别和使用场景:

| PP 值        |   色彩模式   | 描述 |
| ------------ | ------- | --- |
| PP1 | Movie 曲线 + Movie 色彩模式 | 适合直出视频，对比度、饱和度适中，后期调色空间小，不需色彩还原即可进行微调 |
| PP2 | Still 曲线 + Still 色彩模式 | 同上，只是色彩饱和度比较高，整体偏暗，在提高亮部细节的话会牺牲掉暗部细节 |
| PP3 | ITU709 曲线 + Pro 色彩模式 | 同上，但对比度下降，细节大量提升，色彩饱和度降低 |
| PP4 | ITU709 曲线 + ITU709 矩阵色彩模式 | 同上，与 PP3 对比度一样，动态范围不变，色彩饱和度提升 |
| PP5 | Cine1 曲线 + Cinemo 色彩模式 | 与 PP3、PP4 对比，对比度降低很多，色调模拟电影质感 |
| PP6 | Cine2 曲线 + Cinemo 色彩模式 | 同上，区别在于高光部位的细节提升 |
| PP7 | S-log2 曲线 + S-Gamut 色彩模式 | 与 PP5、PP6 对比，对比度更低，高光暗部细节大量提升，色彩饱和度大量降低，套用色彩还原 LUT 后可进行二次微调，整个画面呈现灰色 |
| PP8 | S-log3 曲线 + S-Gamut3.Cine 色彩模式 | 同上，但对比度更低，高光暗部细节大量提升，色彩饱和度微微降低 |
| PP9 | S-log3 曲线 + S-Gamut3 色彩模式 | 与 PP8 类似 |
| PP10 | HLG2 曲线 + BT.2020 色彩模式 | 在保留大量暗部亮部细节的基础上，对比度比 S-Log 高，亮度比色彩饱和度也比 S-Log 高，不需要色彩还原即可进行二次微调 |

> 想要直出视频最好关闭 PP 值或者选择 PP4 之前的设置，因为之后的设置基本要用后期进行调色处理。若 PP10 拍摄出来视频在显示器上出现过曝的情况，是因为 SDR 显示器显示不了 HDR 的色彩，需要套用`效果 -> 色彩 -> HDR 工具`来显示。

> LUT 全称 Lookup Table，即颜色查找表，通过 LUT，你可以将一组 RGB 值输出为另一组 RGB 值，从而改变画面的曝光与色彩。

上面所说的情况都是默认的，我们也可以单独对 PP 值进行设定，比如可以调节黑色等级、伽马、色彩模式等，我们稍微了解下这些参数:

* 黑色等级 - 可以调节基准黑电平，即调整黑色的最低点，从而达到调整视频的对比度。调节范围为正负 15，越大则暗部细节会被提升，对比度会下降。
* 伽马 - 成像物体形成画面的“反差系数”，如果伽马曲线比较抖，则输出的画面反差就会比较高，反之，则输出的画面反差比较低。如 Movie、S-Log3、Cine1 等
* 黑色伽马 - 指伽马曲线中光线暗的那一部分伽马曲线。调整黑色伽马，就是改变暗部的曲线形态。打个比方，我们把黑色伽马调低，那么暗部就会更暗，画面的反差就会更强。
* 膝点 - 又叫拐点。它的作用是调整图像的高亮度区，减少曝光过度，使图像的亮度保持在摄像机捕捉的范围之内，使物体的亮部能看到更多细节。但是需要提醒的是，在使用了膝点过后，可能会使图像变的“苍白无力”，这是由于改变膝点同时也会改变摄像机的色度信号，导致图像饱和度下降所致。
* 色彩模式 - 如 Movie、Still、Pro、BT.2020 等

下面给出三套模板，分别是 PP1 Cine4，PP8 S-Log3 和 PP10 HLG3，具体可以参考[摄影师洪峰](https://www.bilibili.com/video/av55317625?from=search&seid=6485041874276950517)的这个设置。那么日常适合什么场景使用呢，以及后期要怎么处理，我们可以总结如下:

1. PP1 Cine4 - 一般用在白天、阴天。在后期处理的时候，我们直接套一个如 `AirColor_4k_Cine_AriGold` 的自定义 cine4 lut 效果即可
2. PP8 S-Log3 - 一般拍摄人物使用，画面比较灰，后期调色的空间比较大。需要注意的是建议过曝拍摄，值为 EV+1.7-2.0，不然后期处理的话会产生很多噪点。在后期处理的时候，需要先把它还原成 Rec.709 色彩模式，可以套用官方的 `SLog3SGamut3.CineToLC-709` lut，之后可以套用其他风格的 lut 进行调色
3. PP10 HLG3 - 一般用在白天，需要拍摄高光和暗部细节的时候用，在我们预览的时候会出现过曝的情况，这时候需要设置一下色彩空间，在视频信息面板左下角，可以选择设置(Setting)，然后在面板上的颜色空间覆盖(Color Space Override)中选择 `Rec: 2020`，这样画面就不会过曝显示了。在套用自定义 lut 的时候，也要将输入改为 `Rec: 2020`

## 镜头及参数

### 变焦 / 定焦

* **定焦**镜头 - 是指只有一个固定焦距的镜头，只有一个焦段，或者说只有一个视野。定焦镜头没有变焦功能。定焦镜头的设计简单，对焦速度快，成像质量稳定。但缺点是需要改变拍摄远近的话，只能够通过摄影人员自己移动来实现。比如 24mm、35mm、50mm 等
* **变焦**镜头 - 在不改变拍摄距离的情况下，可以通过变动焦距来改变拍摄范围，因此非常有利于画面构图。缺点是光圈普遍偏小，价格会更昂贵，体积较大，不如同焦段定焦镜头成像素质高。广角焦段比如 11-24mm、14-24mm、16-35mm 等等；标准焦段比如 24-70mm、24-85mm、24-105mm 等等；长焦焦段比如：70-200mm、70-300mm、100-400mm 等等。

### 参数解析

1. E-mount ／ A-mount
   1. A-mount(A 接环) 是索尼早期推出独特的半透明式反光镜机身所采用的系统
   2. E-mount(E 接环) 则是无镜式机身所采用的系统，如 A7 系列
2. E 镜头 / FE 镜头
   1. E 表示非全画幅镜头
   2. FE 则表示全画幅镜头。所以在买镜头前，你必需知道自己的相机是「全画幅」或是「非全画幅」
3. 镜头焦距 - 在 E 或 FE 之后，你会看到类似 16-35 这种数字，有些会在其后紧接着一个单位(mm)，这就是显示镜头的焦距。这个例子中，16mm 便是镜头的最短焦距，35mm 是最长焦距，代表这是一支变焦镜头。有些镜头只会写一个数字，如 50，则此镜头为一支定焦镜头。
4. 最大光圈 - 在镜头焦距旁，你会看到像「3.5-6.3」或「2.8」这样的字样。前者通常能在变焦镜头上看到，代表在最短焦距时，镜头的最大光圈为 F/3.5；最长焦距时的最大光圈为 F/6.3。后者，则在定焦或变焦镜头也有机会看到，而图中例子就正是「2.8」，代表镜头有着的是恒定光圈，最大光圈永远是 F/2.8。
5. 镜头直径 - 如果你在镜头正前方看到一个圆形被一斜线穿过的符号，后面又跟随着一个双位数字，即代表那是镜头直径。知道镜头直径后，你就可以购买适合的滤镜转接环及滤镜。
6. 最短对焦距离 - 在镜头正前方或镜身，你应该会看到类似的一组字样，这代表着镜头的最短对焦距离，即相片主体与相机机身上「Φ」之间的最短距离。在这个例子 0.5m/1.64ft 中，即代表机身上的「Φ」符号，至相片中的主角，必需至少保持 0.5 米或 1.64ft 的距离，才会合焦。
7. G / GM / ZEISS / OSS
   1. 橙底白 G 字 - 索尼所认证的高级镜头，不管在解析度、对比度、散景﹑自动对焦及镜身造工全部都是最好﹑最顶级的 G Master
   2. 黑底白 G 字 - 这系列都是自家推出的准专业级镜头和消费级视象镜头，价格比 G Master 较低
   3. ZEISS - 是索尼与蔡司合作，一同研发的一系列专业镜头
   4. OSS - 光学防抖功能

> 三轴指的是平移轴、俯仰轴和横滚轴，三个轴对应有三个锁扣来固定

下面记录一些常用的按键模式:

1. 电源按键
   1. 长按 - 开关机
   2. 短按 - 开启/关闭休眠
2. M 按键
   1. 短按 - 按顺序选择三种预设模式，模式可以在 Ronin APP 的用户参数中设置，分别有`全跟随、平移跟随、FPV、3D Roll 360 和自定义等`
      1. 全跟随 - 平移和俯仰轴开启跟随
      2. 平移跟随 - 平移轴开启跟随
      3. FPV - 三轴皆开启跟随
      4. 3D Roll 360 - 可以 360 度旋转拍摄。使用该功能时，在 Ronin App 用户参数页面的摇杆设置内可设置摇杆最大速度和平滑度，用户参数页面速度、死区参数不起作用
   2. 长按不放 - 开启**运动(疯狗)模式**，配置灯变为黄色，取消长按可退出。若要保持运动模式，可以长按不放并双击扳机键，再次操作可退出。主要用于跟随高速运动的拍摄主体
   3. 三击 - 快速进入/退出 3D Roll 360 模式
   4. 六击 - 恢复蓝牙密码
3. 相机控制键 - 前提是使用相机控制线连接云台相机控制接口与相机
   1. 半按 - 对焦
   2. 短按 - 录像
   3. 长按 - 拍照
4. 扳机键(模式切换键)
   1. 长按 - **全锁定模式**，模式绿灯闪烁
   2. 双击 - 云台回中
   3. 三击 - 自拍模式
   4. 四击且第四击长按 - **摇杆校准**。当云台出现漂移(没有通过摇杆控制云台但云台自行运动)才需要进行摇杆校准。启动摇杆校准后，指示灯循环闪烁，校准过程中持续最大幅度推动摇杆转圈。转动摇杆数圈之后，重复启动校准动作可结束校准，或 20 秒后自动结束校准。如果指示灯出现红灯常亮，则表示校准失败，需要重新校准。
   5. 与 M 键一起长按 4s - **自动校准**。校准前请确保三轴锁扣均打开。通过 Ronin APP 也可以点击自动校准

> 运动模式使云台从慢速跟随切换到快速跟随的模式，云台自动将跟随速度提高到最大，**死区**响应调整至 0 以提高跟随速度。死区设置决定云台将忽略的角度转动量，当如影的转动量超过对应轴的死区设置时，如影才会开始跟随动作。更多如影 SC 用户操作手册[可以查看官网](https://www.dji.com/cn/ronin-sc/info#downloads) 👈
