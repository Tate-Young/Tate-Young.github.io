# 关于这篇博客的描述

此博客诞生于 2018 年 1 月 1 日，自己捣鼓了下分页、评论等，然后加入了 Jekyll 模板。里面所记录的内容都是按照自己理解思路来的，也有转载一些大大的分析，总体而言肯定有很多理解不透彻或不对的地方，还望帮我指正，一起学习。**学无止境~**
